# Fubuki Plasma
My personal KDE Plasma setup inspired by Shirakami Fubuki.

## **⚠️ WARNING**
- Arch Linux only
- KDE Plasma session required
- The installer modifies your oh-my-zsh, zsh and Plasma configurations and may overwrite existing settings
- The installer is experimental. If nothing changes after relogin, run it again

## Installation
### 1. Install my scripts
```bash
curl -fsSL https://raw.githubusercontent.com/sakievmi-dev/fubuki-plasma/main/install.sh | sudo bash
```

### 2. Apply the setup
```bash
fubuki-install
```

## Keybindings cheatsheet
This setup includes a `fubuki-cheatsheet` script that shows all custom keybindings that i use.

```bash
fubuki-cheatsheet
```
